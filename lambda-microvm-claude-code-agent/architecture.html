Lambda MicroVM Claude Code Agent

Long-running Firecracker MicroVM with the Claude Code CLI — Bedrock-backed, shell-accessible, IAM-scoped AWS access

Security & Identity

  • • No secrets baked into the image — only non-secret config (region, model id)
  • • Execution role supplies all credentials via the boto3 default chain
  • • Two read-only layers: IAM ReadOnlyAccess + MCP READ_OPERATIONS_ONLY
  • • Firecracker VM isolation contains the blast radius
  • • SHELL_INGRESS shell access gated by a 30-min auth token

Runtime & Lifecycle

  • • Image built server-side from a Dockerfile, then snapshotted
  • • Resumes from snapshot in under a second
  • • Auto-suspends after 1h idle; auto-resumes on demand
  • • Terminates after 30 min in suspended state
  • • Lifecycle hooks on :9000 · health endpoint on :8080

AI & AWS Access

  • • Claude Code CLI baked in via the native installer (no Node.js)
  • • Amazon Bedrock — Claude Haiku 4.5
  • • US cross-region inference profile (Haiku unavailable in-region)
  • • AWS API MCP server turns natural language into validated API calls
  • • Project-scope .mcp.json pre-approved in /workspace
